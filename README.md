# La Cappuccina ☕

## Objetivo

O objetivo deste projeto é facilitar o processo de pedidos da cafeteria fictícia **La Cappuccina**, desenvolvendo uma solução inovadora e prática para a empresa. O site busca atrair novos consumidores e fortalecer a credibilidade do negócio por meio de uma página única que integra a história da cafeteria, um cardápio completo e depoimentos de clientes, oferecendo uma experiência simples, organizada e agradável.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura das páginas/seções

O projeto segue uma organização modular para facilitar a manutenção:

- `index.html`: página principal de entrada.
- `pages/`: contém as subpáginas do site (`historia.html`, `pedidos.html` e `contato.html`).
- `javascript/`: scripts responsáveis pela interatividade, como criação de pedidos, ícones e lógica de pedidos.
- `.public/`: armazenamento de ativos estáticos, como imagens e banners.
- `style.css`: arquivo centralizado de toda a identidade visual.

## Funcionalidades implementadas

- **Carregamento dinâmico de pedidos:** visualização das solicitações em tempo real.
- **Criação de novos pedidos:** interface funcional para o usuário realizar escolhas.
- **Sistema de ícones:** implementação de elementos visuais para melhorar a navegação e a estética do site.
